The Life of Pablo

  • HTML
  • CSS
  • jQuery

An experimental website that unpacks a vinyl record inside the browser. The goal was to rebrand Mr. West's album which lacked any reference to famous "Pablos." I used SVG paths to wrap the album's song title around the vinyl's coaster.
